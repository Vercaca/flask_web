Variable Types: dict, list, instance, class

mydict["key"] = {{ mydict["key"] }}

mylist[3] = {{ mylist[3] }}

mylist[myintvar](myintvar={{ myintvar }}) = {{ mylist[myintvar] }}

instance.method1() = {{ instance.method1() }}

instance.method2() = {{ instance.method2() }}

instance.method3(5) = {{ instance.method3(5) }}

class.method2() = {{ class.method2() }}

class.method3(10) = {{ class.method3(10) }}

Jinja2 does not support all python syntax

mylist length: { { len(mylist) } } <-- not work

Variables can be modified with filters(pi>pe character as separator)

mylist length: {{ mylist|length }}

code: {{ code }}

code|safe: {{ code|safe }}

code|striptags: {{ code|striptags }}

-- Common filters

    safe: Renders the value without applying escaping
    capitalize: Converts the first character of the value to uppercase and the rest to lowercase
    lower: Converts the value to lowercase characters
    upper: Converts the value to uppercase characters
    title: Capitalizes each word in the value
    trim: Removes leading and trailing whitespace from the value
    striptags: Removes any HTML tags from the value before rendering
